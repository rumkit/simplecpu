Здоров, теперь у нас есть простой калькулятор. Но как теперь с помощью всего этого сделать компьютер?


Конечно, при помощи кода.

После того, как вы узнали, что такое данные, будет очень легко понять,
что из себя представляет код. Это просто данные. Причиной, по которой
код выделяется из остальных данных, является то, что процессор
интерпретирует эти данные как инструкции

Предположим как могла бы выглядить система таких инструкций для произведения компьютером ариметических действий;

инструкция код
"добавь число к другому числу"    00000001
"вычти число из другого числа"    00000010

При помощи этого, мы можем определить, какие логические вентили будут использованы.

Вся память компьютера разделена на адреса и позволяет ЦПУ запрашивать щначение из памяти по определенному адресу. Адрес также может быть представлен числом в двоичном виде. Таким образом, если мы хотим получить значения из памяти по адресу "пять", необзодимо обратиться по адресу "00000101".
Добавим еще одну команду в нашу таблицу:

"Перенеси эти данные в другое место"    00000011

Здорово, теперь мы можем скомандовать:

  "добавь к числу по адресу #5 в памяти другое число по адресу
  #7 в памяти."

Записав это в виде команд:

  (добавить) (адрес памяти #5) (адрес памяти #7)

И в виде кода:

  00000001 00000101 00000111

Довольно просто, не так ли?
