Вот (в первом приближении) простой ЦПУ, чтобы вы могли поиграться с ним;

Чтобы изменять значения в памяти, просто нажимайте на биты пока ЦПУ выключен. Вы также можете прокрутить вниз значения памяти. Чтобы запустить ЦПУ и начать выполнение инструкций, включите переключатель.

У ЦПУ есть три различные инструкции:
Инструкция Код
Прибавить значение, хранящееся по адресу
(Указатель инструкций + 1), к значению по адресу
(Указатель инструкций + 2)
    00000001
Вычесть из значения, хранящегося по адресу
(Указатель инструкций + 2), значение по адресу
(Указатель инструкций + 1)
    00000010
Переместить значение, хранящееся по адресу
(Указатель инструкций + 1), в ячейку с адресом
(Указатель инструкций + 2),		    00000011
Записать в Указатель инструкции значение по адресу (Указатель инструкций + 1)    00000100
Записать в Указатель инструкции значение по адресу (Указатель инструкций + 1),
если значение по адресу (Указатель инструкций + 2) не 0		    00000101

Все инструкции имеют размер 3 байта (включая операнды). Это означает, что мы увеличиваем значение указателя на инструкцию на 3 после выполнения каждой инструкции.

Указатель на инструкцию находится в памяти по адресу 00000000.
